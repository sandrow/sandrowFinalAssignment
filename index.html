How sintac works...

Sintac is a great new app that works hand in hand with any text editor. Simply run the app in the background and it will mirror everything you do in your text editor, finding all syntax errors as well as spelling mistakes. As a bonus addition to this app you can also preview any changes made live.

Easy syntax correction

With the new sintac app you can say goodbye to those long hours spent looking for that one missing comma. Simply run this application while you're coding and it will mirror everything that you do and when the app encounters a syntax error it will then highlight the error as well as show the exact line the error occurred in the console log.

Simple auto correct

Sintac has a unique feature which surprisingly most text editors don't have and the is auto correct. With this feature you don't have to worry about you website breaking due to a silly typo, sintac will find the the mistake and highlight it green as well as show the exact line the error occurred in the console log

Easy browser preview

Being able to preview all the changes you make as they happen without having to constantly refresh your browsers is extremely beneficial, because it saves you time by allowing you to see your mistakes as they happen.
